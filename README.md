# AI Job Discovery Dashboard

A portfolio project demonstrating a job recommendation engine that surfaces high-quality opportunities across:

• Operations
• Compliance / Risk
• AI / Transformation

Features:
- recommendation logic with lane balancing
- job ranking and diversification
- application tracking pipeline
- recruiter-ready dashboard UI
- deployed using GitHub Pages
